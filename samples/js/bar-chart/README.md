# Plotly Bar Chart

This renders noms data as a bar chart using Plotly (https://plot.ly/javascript/).

![bar chart](screenshot.png)

## Build

```
pushd ..
npm install
popd
npm install
```

## Run

```
noms ui .
```

Then, navigate to the URL printed by `noms ui`, e.g. http://127.0.0.1:12345

## Develop

```
npm run start
noms ui .
```
